**📁 Location**: `react-native/`

**📱 Platforms**: iOS & Android
**⚡ Features**:
- Native mobile browser with WebView
- Touch-optimized interface
- AI search and Web3 integration
- Bookmarks and history management

### **Build Mobile Apps:**

#### **Android APK:**
```bash
cd react-native
npm run android        # For development
npm run build:android  # For production APK
```

#### **iOS App:**
```bash
cd react-native
npm run ios            # For development (requires Xcode)
npm run build:ios      # For production (requires Xcode)
```

### **Mobile Development Setup:**
1. Install React Native CLI: `npm install -g react-native-cli`
2. For Android: Install Android Studio and Android SDK
3. For iOS: Install Xcode (macOS only)
4. Run `npm start` to start Metro bundler
5. Run `npm run android` or `npm run ios` to launch

**🌐 Platforms**: Chrome, Edge, Firefox (with minor changes)
**⚡ Features**:
- AI page summarization on any website
- Web3 wallet balance checker
- Content analysis tools
- Context menu integration

### **Install Browser Extension:**
1. Open Chrome/Edge
2. Go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the `browser-extension/` folder
6. Extension will be installed and ready to use
